# Full Stack Developer — Screening Task

Welcome! This task has **3 parts** and should take around **2-3 hours total**.

---

## Part 1 — Code Review (30 mins)

Open `part1_review_this_code.ts`. It has **at least 4 issues** — a mix of bugs and bad practices.

Your task:
- Find and fix all the bugs
- Identify the bad practices and explain why they're bad
- Add short comments explaining each fix

Do NOT rewrite the entire file — just fix what's broken.

---

## Part 2 — Work on an Existing Codebase (90 mins)

The `part2_codebase/` folder contains a Client Appointment System (TypeScript + Express). Read `part2_build_endpoint.md` for full instructions. In short:
- **Fix a bug:** the system allows double-booking overlapping appointments
- **Add a feature:** build a daily schedule endpoint that shows appointments, free slots, and a summary
- Work with the existing code — don't rewrite it

### Setup
```bash
cd part2_codebase
npm install
npx ts-node server.ts
```

---

## Part 3 — Written Question (15 mins)

Read `part3_written_question.md` and answer in your own words (5-10 sentences).
